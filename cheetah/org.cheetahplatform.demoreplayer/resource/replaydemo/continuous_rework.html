Measures

Our measures classify this process as showing a low chunk size (3) with a medium number of iterations (18). Moreover, the measures show a high number of reconciliation breaks (38.5%), i.e., modeling iterations that show unexpected reconciliation. This example shows an average score with respect to time spent on comprehension (56.6%).

Modeling Strategy

The replay confirms the observation that the model is created very slowly; adding new elements is directly followed by layouting, i.e., a reconciliation phase. Extensive layouting also happens prior to adding new elements in order to prepare a certain area of the model for adaptation. Furthermore, reconciliation often affects large parts of the existing model beyond the elements added last. We characterize this strategy as 'continuous rework'.
